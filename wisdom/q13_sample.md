# 教學智慧筆記：切線速度與角速度的轉換誤區

## 題目
ID: q13
題目文字: 兩個星體 A 與 B 以角速度 $\omega$ 繞質心公轉。若 A 距離質心 $R_A$，B 距離質心 $R_B$，且 $R_A = 2 R_B$。請問兩星體的切線速度大小比例 $v_A : v_B$ 為何？
選項:
- A: $v_A : v_B = 1:2$
- B: $v_A : v_B = 2:1$
- C: $v_A : v_B = 1:4$
- D: $v_A : v_B = 4:1$
正確答案: B

## 老師的診斷智慧
- 如果學生選 A，他可能記反了公式，或者是把 $v = \omega r$ 以為是 $v = \omega / r$。
  - 提示 1: 想想看，在同樣的轉動時間內，外圈的人跑的路徑比較長還是內圈的人？
  - 提示 2: 切線速度的公式是 $v = \omega r$，當 $\omega$ 相同時，$v$ 與 $r$ 是成正比還是反比？
- 如果學生選 C 或 D，他可能把「加速度」或「角速度平方」的關係帶進來了，或者是把 $r$ 給平方了（像是在帶萬有引力公式）。
  - 提示: 這裡只問速度 (v)，不是加速度 (a)，不需要把半徑平方。
- 如果有人選了奇怪的數字，那通常是單純的代數計算錯誤。

## 推薦路徑
- 如果連 $v = \omega r$ 的正比關係都弄錯，回頭去練「基礎圓周運動公式 (q2)」。
- 如果對了，就去算「角動量 $L = mvr$ 的進階應用題 (q11)」。
