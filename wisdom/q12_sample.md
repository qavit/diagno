# 教學智慧筆記：雙星系統的動量守恆與方向性

## 題目
ID: q12
題目文字: 在一個互繞運行的雙星系統中，恆星 A 的質量為 M，恆星 B 的質量為 3M。請問在「質心坐標系 (Center-of-Mass Frame)」中，這兩顆恆星的動量大小 (Magnitude) 的比例為何？
選項:
- A: $P_A : P_B = 1:3$
- B: $P_A : P_B = 3:1$
- C: $P_A : P_B = 1:1$
- D: 無法確定，因為沒有給予軌道半徑
正確答案: C

## 老師的診斷智慧
- 如果學生選 A，他可能被「$P = mv$」帶著走了，因為質量比是 1:3，他直接以為動量比也是 1:3。
  - 提示 1: 想想看「質心坐標系」定義下的總動量應該是多少？
  - 提示 2: 既然總動量是 0，代表兩者的動量向量之和必須為零。
- 如果學生選 B，他可能是想到了「反比」，但他把反比用錯地方了（速度才是反比，動量大小應該相等）。
  - 提示: 速度 $v$ 的確跟質量成反比，但動量 $P = mv$ 呢？
- 如果學生選 D，他對質心坐標系的物理本質還不夠熟悉，以為需要更多幾何資訊。
  - 提示: 質心坐標系有一個強大的特性，跟軌道半徑無關。

## 推薦路徑
- 如果答錯（尤其是選 A），讓他去寫「質心坐標系的基本概念題 (q3)」。
- 如果答對，讓他去算「綜合題：動量與角動量的合集 (q10)」。
